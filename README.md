🧠 Quiz Masters
A fun and interactive Tech Quiz Game built with HTML, CSS, and JavaScript.
✨ Features:
20 Logical Questions in English.
Timed Challenges for each level.
Level System: Smooth transition from Level 1 to Level 2.
Audio Feedback: Sounds for correct and wrong answers.
Score Review: See your performance at the end.
🛠️ Built With:
Languages: HTML5, CSS3, JavaScript
Editor: Spck Editor (Mobile)
